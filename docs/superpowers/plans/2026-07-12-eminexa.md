# Eminexa Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build **Eminexa** — the people-connection subsystem of the Manifexa engine: you open `manifexa` on an **Eminexa data folder**, seed a researcher from the interactive shell, and the engine fetches their 5-year record from OpenAlex and grows a graph of `person` nodes joined by `coauthored` and inferred `same_group` edges. Eminexa is a *folder you keep people in*, not a separate binary or module.

**Architecture:** Eminexa reuses the engine's existing "**cache → derived graph**" spine. People + edges are written to the SQLite `cache` (persistent, owned) with rich fields in the node `meta` blob; the existing `App.rebuild()` (`app.py:49`) projects the cache into ArcadeDB on every write (confirmed at `graph/sync.py:33-50`: every cache node projects as a `candidate`, every cache edge carries its `rel`); and the existing `discovery/core.py` (`around`/`path`/`bridges`/`clusters`) surfaces them for free. No new graph backend, no `clear()` surgery — the cache already survives rebuilds. Everything is deterministic and OpenAlex-only; the web/Scholar current-snapshot layer is explicitly deferred (Phase 2).

**Command surface:** All Eminexa commands live in the **interactive shell** — the single `tui.dispatch(app, line, st)` dispatcher (`tui.py:647`), NOT the argparse `cli.py`. The clean, people-first verb set (approved 2026-07-13):

| group | verb | does |
|---|---|---|
| grow | `add <id·orcid·scholar>` | add a person + their coauthors (routes person-seeds to `add_person`; other seeds fall through to the existing paper/create `add`) |
| grow | `expand <id>` | flesh a coauthor stub into a full node (routes person-seeds to `add_person`; else the existing LLM `expand`) |
| see | `who <id>` | a person's card — prominence, topics, affiliations, coauthors |
| see | `list` · `map` | everyone / the whole graph (existing) |
| connect | `near <id>` | alias of `around` |
| connect | `path <a> <b>` · `bridges` | existing |
| connect | `groups` | alias of `clusters` (the inferred `same_group`) |
| keep | `save`/`load` | aliases of `export`/`import` |
| keep | `drop <id>` | alias of `remove` |
| meta | `help` · `color` · `quit` | existing |

Aliases and person-seed routing are **additive** — no existing command's behavior is removed. The polished surface is achieved by (a) adding the new verbs, and (b) rewriting the visible `HELP`/palette to show only the clean set (Task 8); the demoted paper/LLM/vault commands keep working but are undocumented. Full removal of demoted commands is deferred pending user sign-off (was "remove vs hide" — hide chosen for v1: reversible, ships fast).

**Tech Stack:** Python 3.12+ (stdlib `urllib` for HTTP, as the codebase already does), `pytest`, existing `manifexa.store.cache.Cache`, `manifexa.sources.openalex.OpenAlexClient`, `manifexa.discovery.core`, NetworkX Louvain (already wired in `clusters()`).

## Global Constraints

- **Never commit credentials.** The OpenAlex API key lives in `~/.manifexa/openalex.json` and must never enter git or logs.
- **OpenAlex is the only source in v1.** No web scraping, no Scholar fetch, no balthar — those are Phase 2.
- **Throttle OpenAlex:** ≤ 10 requests/sec with backoff on HTTP 429 (we hit sustained 429s during design). Every OpenAlex call in Eminexa goes through the throttled client.
- **People live in the cache, not the vault.** Eminexa never calls `schema.validate` — cache nodes are schemaless (`key,type,title,meta,source`). Store all rich fields in the node `meta` dict. Tag every Eminexa write with `source="eminexa"` so `App.forget("eminexa")` can drop the whole people-graph without touching curated files.
- **5-year window** is relative to the run date, passed in explicitly (never `datetime.now()` inside pure functions — pass `today` as an argument so tests are deterministic).
- **Tests are offline.** Inject a fake OpenAlex client (a small object returning recorded dicts); never hit the live API in tests. Follow the existing pattern (`enrich_seed(client, …)` takes an injected client).

# manifexa/eminexa/clean.py
"""Cleaning guards — normalize scalar strings and dedup near-duplicate works
BEFORE any coauthor facts are computed (duplicate work records inflate counts)."""
from __future__ import annotations

import html
import re
import unicodedata


def normalize(s: str) -> str:
    """HTML-unescape, NFKC-fold, collapse whitespace, casefold. Deterministic."""
    if not s:
        return ""
    s = html.unescape(s)
    s = unicodedata.normalize("NFKC", s)
    s = re.sub(r"\s+", " ", s).strip()
    return s.casefold()


def _doi_norm(doi) -> str:
    return normalize((doi or "").replace("https://doi.org/", ""))


def _title_key(w: dict) -> str:
    t = normalize(w.get("title") or "")
    t = re.split(r"[-–]\s*suppl", t)[0]          # drop "- Supplementary ..."
    return re.sub(r"[^a-z0-9]+", " ", t).strip()


def _cluster_tkey(w: dict) -> str:
    return _title_key(w) + "|" + str(w.get("publication_year"))


def dedup_works(works: list[dict]) -> list[dict]:
    """Collapse duplicate work records to one canonical per paper. Two passes so a
    DOI-less copy still joins its DOI-bearing twin: (1) map each (title, year) that
    has any DOI to that DOI; (2) cluster by DOI when the record has one (or its
    title-cluster does), else by (title, year). The DOI-bearing record wins."""
    title_doi: dict[str, str] = {}
    for w in works:
        d = _doi_norm(w.get("doi"))
        if d:
            title_doi.setdefault(_cluster_tkey(w), d)
    best: dict[str, dict] = {}
    for w in works:
        own = _doi_norm(w.get("doi"))
        d = own or title_doi.get(_cluster_tkey(w), "")
        key = ("doi:" + d) if d else ("t:" + _cluster_tkey(w))
        cur = best.get(key)
        if cur is None or (own and not _doi_norm(cur.get("doi"))):
            best[key] = w
    return list(best.values())
```

# manifexa/eminexa/node.py
"""Build the locked Eminexa `person` node from an OpenAlex author + their window
works. Everything is derived from the 5-year window (sharper than lifetime author
fields); lifetime prominence (works/cites/h) is kept as a cheap seniority signal."""
from __future__ import annotations

import collections

from .clean import dedup_works
from ..sources.openalex import normalize_openalex_id


def build_person_node(author: dict, works: list[dict], today_year: int) -> dict:
    aid = normalize_openalex_id(author["id"])
    works = dedup_works(works)

    topics = collections.Counter()
    affs = collections.Counter()
    coauth: dict[str, dict] = {}
    ids = []
    for w in works:
        ids.append(normalize_openalex_id(w["id"]))
        yr = w.get("publication_year") or 0
        for t in (w.get("topics") or [])[:3]:
            if t.get("display_name"):
                topics[t["display_name"]] += 1
        for sh in w.get("authorships", []):
            au = sh.get("author") or {}
            k = normalize_openalex_id(au.get("id") or "")
            if not k:
                continue
            if k == aid:
                for s in (sh.get("raw_affiliation_strings") or []):
                    affs[s] += 1
            else:
                c = coauth.setdefault(k, {"id": k, "name": au.get("display_name", ""), "n_shared": 0, "last_year": 0})
                c["n_shared"] += 1
                c["last_year"] = max(c["last_year"], yr)

    orcid = (author.get("orcid") or "").replace("https://orcid.org/", "") or None
    alts = [x for x in (author.get("display_name_alternatives") or []) if x != author.get("display_name")]
    meta = {
        "openalex": aid,
        "orcid": orcid,
        "aliases": alts,
        "works_count": author.get("works_count"),
        "cited_by_count": author.get("cited_by_count"),
        "h_index": (author.get("summary_stats") or {}).get("h_index"),
        "window_work_ids": ids,
        "topics": [t for t, _ in topics.most_common(8)],
        "affiliations": [a for a, _ in affs.most_common(5)],
        "coauthors": sorted(coauth.values(), key=lambda c: -c["n_shared"]),
        "source": "eminexa",
    }
    return {"key": aid, "type": "person", "title": author.get("display_name", ""), "meta": meta}
```

# manifexa/eminexa/people.py
"""Author-centric OpenAlex fetches with a polite throttle (≤10 req/s) and 429
backoff. Wraps an OpenAlexClient (or any object exposing `_get(path, params)`)."""
from __future__ import annotations

import time
import urllib.error

from ..sources.openalex import normalize_openalex_id

_MIN_INTERVAL = 0.12          # ~8 req/s, under OpenAlex's 10/s
_last = [0.0]


def _throttle():
    dt = time.monotonic() - _last[0]
    if dt < _MIN_INTERVAL:
        time.sleep(_MIN_INTERVAL - dt)
    _last[0] = time.monotonic()


class PeopleClient:
    def __init__(self, oa, tries: int = 4) -> None:
        self._oa = oa
        self._tries = tries

    def _get(self, path, params=None):
        for i in range(self._tries):
            _throttle()
            try:
                return self._oa._get(path, params or {})
            except urllib.error.HTTPError as e:
                if e.code == 429 and i < self._tries - 1:
                    time.sleep(2 * (i + 1))
                    continue
                raise

    def get_author(self, author_id) -> dict:
        return self._get(f"authors/{normalize_openalex_id(author_id)}")

    def works_by_author(self, author_id, from_date: str, cap: int = 200) -> list[dict]:
        aid = normalize_openalex_id(author_id)
        sel = "id,title,doi,publication_year,topics,authorships"
        return self._get("works", {"filter": f"author.id:{aid},from_publication_date:{from_date}",
                                    "sort": "publication_date:desc", "per_page": str(cap),
                                    "select": sel}).get("results", [])

    def author_by_orcid(self, orcid) -> dict | None:
        r = self._get("authors", {"filter": f"orcid:{orcid}", "per_page": "1"}).get("results", [])
        return r[0] if r else None
```

# manifexa/eminexa/resolve.py
"""Resolve a seed to an authoritative OpenAlex author id — never by bare name.
v1 seeds: OpenAlex author id (A…), ORCID (0000-…). Scholar URL + paper-anchor
are Phase 2."""
from __future__ import annotations

import re

_ORCID = re.compile(r"^\d{4}-\d{4}-\d{4}-\d{3}[\dX]$")


def resolve_seed(pc, seed: str) -> str:
    s = seed.strip()
    if re.fullmatch(r"A\d+", s):
        return s
    if _ORCID.match(s):
        a = pc.author_by_orcid(s)
        if a:
            return a["id"].split("/")[-1]
        raise ValueError(f"no OpenAlex author for ORCID {s}")
    raise ValueError(f"unrecognized seed '{seed}' — use an OpenAlex author id (A…) or an ORCID")
```

# manifexa/eminexa/ingest.py
"""Ingest one person into the cache: person node + coauthor stubs + `coauthored`
edges, all source="eminexa". The existing App.rebuild() projects these into the
graph; discovery (clusters/bridges/around) then works on people for free."""
from __future__ import annotations

from .node import build_person_node
from .resolve import resolve_seed


def ingest_person(cache, pc, seed: str, today_year: int, from_date: str) -> dict:
    aid = resolve_seed(pc, seed)
    author = pc.get_author(aid)
    works = pc.works_by_author(aid, from_date=from_date)
    node = build_person_node(author, works, today_year=today_year)

    cache.upsert_node(node["key"], "person", node["title"], node["meta"], source="eminexa")
    edges = 0
    for c in node["meta"]["coauthors"]:
        if not cache.get_node(c["id"]):
            cache.upsert_node(c["id"], "person", c["name"], {"openalex": c["id"], "stub": True}, source="eminexa")
        cache.upsert_edge(node["key"], c["id"], "coauthored", source="eminexa")
        edges += 1
    return {"person": node["key"], "coauthors": len(node["meta"]["coauthors"]), "edges": edges}
